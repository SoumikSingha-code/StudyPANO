How To Make Notes?





1) Understand the purpose of notes Notes are not meant to copy the book. Their job is to:

  • help you understand concepts
  • make revision faster
  • highlight important points
  • store information in your own words so it stays in memory longer
Good notes = short + clear + meaningful.

2) Read first, then write Never start writing immediately. Step-by-step:

Read the topic once to understand the idea.
Read again and find:

  • definitions
  • formulas
  • dates
  • keywords
  • diagrams
Now write notes in your own words. This improves understanding and memory.

3) Use simple structure Always keep notes organized. Use this order:

(a) Heading

Write the chapter/topic name clearly.

(b) Subheadings

Break into parts like:
  • Definition
  • Types
  • Formula
  • Examples
  • Diagram
  • Uses

(c) Bullet points

Avoid long paragraphs. Use points.
Bad:
Photosynthesis is a process in which plants prepare food using sunlight...

Good:
Process by which plants make food Using sunlight, water, carbon dioxide and Produces glucose and oxygen

4) Use keywords, not full sentences Write only important words.

Example:

Instead of:
The French Revolution started in 1789 due to social inequality.
Write:
      French Revolution – 1789
      Cause: social inequality

This saves time and improves revision speed.

5) Use diagrams and flowcharts

Especially useful for:

  • Science
  • Geography
  • Biology
  • Computer

    • Examples:

  • Life cycle
  • Process steps
  • Ray diagrams
  • Maps

Visual memory is stronger than text memory.

6) Highlight important things and Use:

  • boxes
  • highlighter
  • underlining

different colors Highlight:

  • formulas
  • dates
  • definitions
  • keywords

    • 7) Make subject-wise style










    Different subjects need different note styles.

    Science :
    • definitions
    • formulas
    • diagrams
    • steps of experiments
    Maths :
    • formula list
    • solved examples
    • important tricks
    History / Geography :
    • dates
    • events
    • causes & effects
    • maps
    Computer / Programming :
    • syntax
    • commands
    • small code examples
    • error notes

    8) Use the “Question method”.Write notes like answers.

    Example:

    Q: What is refraction?
    Ans: Bending of light when it passes from one medium to another. This helps directly in exams.

    9) Keep notes short for revision Make two types:

    • Detailed notes (for understanding)
    • Quick revision notes (1–2 pages per chapter) Before exams, only read revision notes.

    10) Revise and update notes After class:

    • add teacher’s explanation
    • add examples
    • correct mistakes
    • improve clarity
    • Notes improve over time.

    11) Digital vs handwritten notes

    Handwritten
    • better memory
    • better for school exams
    • diagrams easy
    Digital
    • easy editing
    • searchable
    • good for computer subjects

    Best method: use both.

    12) Common mistakes to avoid :

    • copying book word-to-word
    • writing long paragraphs
    • not using headings
    • not revising notes
    • writing messy notes

    13) Ideal note-making method (topper strategy)

    • Read chapter
    • Understand concept
    • Write short notes
    • Add diagrams
    • Highlight key points
    • Make revision sheet
    • Revise weekly

    14) Powerful techniques students use Cornell Method Page divided into:





    • notes
    • keywords
    • summary
    • Mind maps
    • Central topic → branches → subtopics
    • Formula sheets

    All formulas on one page per chapter.

    15) Final rule Best notes are:

    • short
    • clear
    • organized
    • in your own words
    • easy to revise in 5 minutes

    That is what makes exams easier.
